# I-Built-My-Own-Macro-Pad-From-Scratch-with-ESP32-S3
Custom DIY Macro Pad keyboard built from scratch using ESP32-S3 and 2.8" ILI9341 TFT display. Features Wi-Fi live weather updates, NTP time sync, rotary encoder volume controls, and custom macro hotkeys programmed with Arduino IDE and LovyanGFX.
